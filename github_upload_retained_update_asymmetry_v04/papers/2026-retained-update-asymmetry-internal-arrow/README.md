# Retained Update-Asymmetry and the Internal Arrow of Time

Author: Elad Genish  
Status: v0.4 release candidate  

This folder contains the v0.4 release-candidate short paper:

- `retained_update_asymmetry_internal_arrow_short_v04.md` — source Markdown
- `retained_update_asymmetry_internal_arrow_short_v04.pdf` — rendered PDF
- `retained_update_asymmetry_internal_arrow_short_v04.docx` — editable DOCX copy

The paper presents a finite-formal-system account of temporal appearance as retained update-asymmetry. Its claim is intentionally narrow: it isolates a formal regime in which past/present/future-like roles and an internal arrow-like direction become legible when retention, partial irreversibility, admissible-future constraint, and remaining future openness co-occur.

This release does not claim to derive physical time, the thermodynamic arrow, spacetime geometry, quantum measurement, or cosmological boundary conditions.
